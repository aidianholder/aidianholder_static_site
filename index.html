Public Payrolls

I built this largely because no one else -- public or private -- had, and being able to see how my government spends my money is about as fundamental as transparency gets. I thought it ridiculous that an internet search for Yakima government salaries returned nothing but a link to the HR department. Apparently our readers thought similarly; it's been very popular.

The data is stored as locally hosted json files. I'm using the DataTables library for the display. With barely two thousand records total, there's no reason to use a database, and this way the search and paging is lighting fast. That calculation may change once I add records from surrounding towns.

Yakima PD Incident Map

Call data is scraped twice daily and dumped into a PostgreSQL database, then served via a Django-powered REST API. Configurable queries from the front end launch ajax calls to retrieve call data which is then displayed on a Leaflet powered slippy map.

Along with being a steady traffic draw, the data collected gives us store of incident data for later analysis; I was doing hotspot analysis with YPD's data before YPD was.

Crime Statistics

I built this little interactive largely to get up to speed on the new version of D3. It's a cool element that we can drop into any crime story to provide some context.

We used the latest crime rate data from the FBI to illustrate how much crime happens in every part of the Yakima Valley.

First, select a Yakima County law enforcement agency from the left drop down menu. Then select a type of crime or crimes from the right drop down menu to see how your town compares to the state and nation as a whole.

Health Inspections

Every Yakima County health inspection since 2016. We get monthly data updates emailed, upload them to PostgreSQL and the data is served from a Django-powered backend.

There's a number of problems with the data quality, there are editorial considerations that limit how much I can clean it up, and the whole app needs to be rewritten from scratch.

Despite those issues it was the single most popular page on yakimaherald.com in 2018. You can see it in production at yakimaherald.com/inspections.

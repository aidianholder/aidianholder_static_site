Public payroll

I built this because no one else had, and this is the sort of basic transparency that every community needs. The data is just remote json files; with two thousand records per year, there's no reason to use a database, and this way the search and paging is lighting fast.

Ticket hotspots

A cool example of spatial data which shows where drivers are most likely to get a speeding ticket. The slippy map itself is all running off homerolled, self-hosted vector tiles.

Call data is scraped twice daily and dumped into a PostgreSQL database, then served via a Django-powered REST API. An onscreen slider configures an ajax call to retrieve selected data, which is then displayed on a Leaflet-powered slippy map.
